# 🖥️ HomeForPaws Server

Backend server for the HomeForPaws pet adoption platform.

---

# ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT
- Cookie Parser
- CORS

---

# 📦 Packages Used

```bash
express
mongodb
jsonwebtoken
cookie-parser
cors
dotenv
```

---

# ⚙️ Environment Variables

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```

---

# ▶️ Run Locally

```bash
npm install
nodemon index.js
```

---
